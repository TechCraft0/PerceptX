# 🧠 PerceptX

> **A unified and extensible perception framework supporting both single-stage and two-stage object detection models — from training to deployment.**  
> 一个统一且可扩展的感知框架，支持单阶段与双阶段目标检测模型的训练与部署。

---

## 🚀 Introduction | 项目简介

**PerceptX** 是一个面向通用目标检测任务的统一感知框架，旨在同时支持 **单阶段 (Single-Stage)** 与 **双阶段 (Two-Stage)** 检测模型的训练、推理和部署。  
该项目的目标是为研究者与工程师提供一个高效、模块化、可扩展的检测平台，帮助快速验证算法、优化模型结构、实现端到端部署。

---

## ✨ Features | 特性

- 🧩 **Unified Framework** — 同时支持单阶段与双阶段检测模型（YOLO, RetinaNet, Faster R-CNN 等）  
- ⚙️ **Modular Design** — 模块化代码结构，清晰的训练与推理流程  
- 🧠 **Extensible Backbone** — 轻松集成多种主干网络（ResNet, Swin, ConvNeXt 等）  
- 🚀 **End-to-End Pipeline** — 从数据加载到部署的一体化流程  
- 🧪 **Flexible Experiment Management** — 易于复现的实验配置与日志系统  
- 🌍 **Future-ready** — 后续将支持语义/实例分割、多模态感知等任务
